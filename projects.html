ReadersRecommended

What it does

ReadersRecommended is a website that allows a user to input a book they like a get a list of suggestions of other books they might like based on that book. It uses searching and sorting algorithms to manipulate a large database of books. The program places thousands of book in an undirected graph based on user ratings of the books and then uses breadth-first search to find the top five books that the user will most likely enjoy.

Software Tools Used

  • Java
  • Linux
  • Graphs

Stat.us

What it does

Inspired by wi-fi connectivity issues at a hackathon, Stat.us allows users to stay up-to-date with current news, even without access to the internet. A user just has to send a text and they will recieve a text back with the current trending topics on Twiter. They then can choose to be texted top tweets from those topics, allowing them to catch up with news quickly.

Software Tools Used

  • Python
  • Twilio API
  • Twitter API

ResQ

What it does

ResQ is a smartwatch app that allows distressed users to let someone know they are in danger. A person can simply press a button on the side of their Pebble watch and instantly, a text will be sent to the contact of their choice, stating their need for help and their location in GPS coordinates.

Software Tools Used

  • Pebble Watch
  • Twilio API

Count BACula

What it does

University students tend to push the limits of their alcohol tolerance without really understanding the consequences. Most Blood Alcohol Concentration (BAC) calculators come off very preachy and are targeted at preventing people from drinking. Count BACula a BAC calculator app that encourages students to drink responsibly but doesn't exaggerate the consequences of their drinking. It calculates the estimated BAC of the user after x many shots and hours and tells them how intoxicated they will be at that level. This allows them to be more aware of where their alcohol limit is.

Software Tools Used

  • Python
  • Flask
